Build: 2026-01-11T21:01:58.649376Z

## Co jest w v4 (zgodnie z Twoimi korektami do v3)
- JPG/PNG → PDF A4 w trybie „pełna strona” (cover, jak iLovePDF / skanery)
- Auto-obrót zdjęć: EXIF (best-effort) + ręczne obracanie ↺/↻ na każdej stronie
- X (usuń) na każdej stronie + „Usuń wszystko” + „Usuń zaznaczone”
- Drag&Drop do dodawania plików (bez otwierania w nowych kartach)
- Workspace edytora stron: po wgraniu PDF widzisz miniatury wszystkich stron (jak iLovePDF) i możesz:
  - zaznaczać/odznaczać,
  - usuwać,
  - przestawiać kolejność (drag&drop),
  - eksportować do PDF A4
- Eksport stron do PNG (pobierane kolejno)

## Deploy
Wrzuć `index.html` + `vercel.json` do repo na GitHub (Vercel zdeployuje automatycznie).
